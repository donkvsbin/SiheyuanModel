# 四合院虚拟导览系统

## 项目概述

本项目是一个基于 WebGL 的 3D 四合院虚拟导览系统，旨在帮助外国留学生通过沉浸式体验学习中国传统建筑文化。用户可以在虚拟四合院中自由探索，与场景中的文化点进行交互，了解四合院的历史、建筑特点和文化寓意。

## 核心目的

为外国留学生提供一个互动式的中国文化学习平台，通过游戏化的探索方式，让学习者在虚拟环境中：
- 直观感受四合院的空间布局和建筑特色
- 了解影壁、垂花门、门槛等传统建筑元素的文化含义
- 体验中国传统文化中的吉祥寓意（如石榴的多子多福）

## 技术架构

### 核心技术栈

| 技术 | 用途 |
|------|------|
| **Three.js** | 3D 场景渲染、模型加载、光照系统 |
| **Rapier3D** | 物理引擎，处理角色移动、碰撞检测 |
| **Vue.js 3** | 前端框架，管理 UI 状态和组件 |
| **GLTF/GLB** | 3D 模型格式，支持 Draco 压缩 |

### 关键功能实现

#### 1. 3D 场景渲染
- 天空盒（CubeTexture）实现环境背景
- 定向光模拟日光，支持时间变化（9-18点）
- 后期处理：GTAO（环境光遮蔽）、Bloom（辉光）、体积光

#### 2. 物理系统
- Rapier 角色控制器（Kinematic Character Controller）
- 胶囊体碰撞器处理角色移动和楼梯攀爬
- 自动步进（Autostep）优化低速楼梯通行

#### 3. 交互系统
- 基于距离的交互点检测
- 对话系统支持多语言（中/英文切换）
- Tips 提示框展示文化知识点

#### 4. 剧情系统
- 线性剧情流程管理
- 场景标记和状态持久化
- 多页对话支持

## 文化知识点

系统包含以下文化交互点：

| 交互点 | 文化内容 |
|--------|----------|
| 门槛 | 高30厘米，象征地位，防止雨水倒灌 |
| 影壁 | 遮挡视线，保护隐私，聚气纳财 |
| 垂花门 | 二门，四合院内外院分界，雕刻精美 |
| 石榴树 | 多籽象征多子多福，中秋食俗 |
| 秋千 | 传统庭院娱乐设施 |

## 运行方式

```bash

```

## 技术亮点

1. **性能优化**：使用 Draco 压缩减少模型加载时间，固定像素比提升渲染性能
2. **物理优化**：针对低速移动优化角色控制器，解决楼梯卡顿问题
3. **视觉效果**：GTAO 增强建模间阴影，体积光营造晨光氛围
4. **跨文化设计**：完整的国际化支持，UI 采用中国传统美学风格
